# Y-Crash-Project
We use the Really Fast Causal Inference (RFCI) algorithm (Colombo et al., 2012) to
estimate a Partial Ancestral Graph (PAG) based on conditional independence tests.
We refine this PAG with domain knowledge to infer undirected edges to formulate a
hypothesized DAG. This method is applied to a subset of 250,000 fatal crash 
observations from National Highway Traffic Safety Administration data between the 
years of 2012 and 2024.
